Created by Avais and Swapneel

About Imaginashion

Imaginashion is a project developed for the web technologies labwork. It focuses on providing an open-source solution to pattern recognition across images. It is free to use both by students and professionals.

We have implemented an image recognition algorithm leveraging machine learning built on top of a Django webapp using jQuery for uploading images.

In order to run the algorithm, users simply need to upload an image and select the option acording to their preference.
